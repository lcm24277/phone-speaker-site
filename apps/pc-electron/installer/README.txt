手机变音箱-PC 安装说明

1. 安装桌面程序后，首次启动前请确认是否已经安装 VB-CABLE。
2. 如果尚未安装，可运行同目录下的 `VB-CABLE-Setup-Assistant.bat`。
3. 若你已将官方 `VBCABLE_Setup_x64.exe` 放入：
   integrations\virtual-audio-driver\vb-cable\vendor\
   安装向导会自动尝试拉起它。
4. 安装成功后，请将 Windows 默认输出或目标应用输出切换到 `CABLE Input (VB-Audio Virtual Cable)`。
5. 再返回手机变音箱-PC，扫描并连接手机即可开始收流播放。
